this is a basically a python script that validates vendor sales data against a fixed set of business rules.
you just feed the script a text file and it instantly tells you exactly which rows are invalid and why. 

the input file must be a plain text or csv file with exactly seven columns per row in the following order of the vendor identifier number, vendor name, year to week code, count of vegan hotdogs, count of meat hotdogs, onions in kilograms and ketchup in litres. each field is checked against a precise rule.
